**AutoFlow Hotkeys** is a lightweight Windows macro recorder that makes automating repetitive tasks easy.  
It records mouse movements, clicks, and keyboard inputs, then plays them back with customizable speed, loops, and hotkeys.  

Created for anyone who wants automation without **bloated software or subscriptions**.  

## Features

- Record **mouse and keyboard actions**
- **Customizable global hotkeys** (Record / Play / Stop)
- Playback speed control: **Fast / Normal / Slow**
- **Loop macros** (once, multiple times, or infinite)
- Save and load macros for later use
- Clean, simple, beginner-friendly GUI
- Works even when the app window is minimized  

---

## Requirements

- Windows 10 / 11  
- Python 3.9+  
- Libraries:
  - `pynput`
  - `tkinter` (built-in with Python)
